## Portal del cliente

### Reservar cita

1. Entra como cliente.
2. Abre `Reservar`.
3. Selecciona `Servicio`.
4. Selecciona `Especialista`.
5. Selecciona `Fecha`.
6. Pulsa un hueco en `Horas disponibles`.
7. Pulsa `Confirmar cita`.
8. Comprueba la redirección a `Mis citas`.

### Solicitar otro horario

1. Abre `Reservar`.
2. Selecciona `Servicio`.
3. Selecciona `Especialista`.
4. Rellena `Fecha y hora deseada`.
5. Rellena `Preferencias`, si procede.
6. Pulsa `Enviar solicitud`.

### Consultar citas propias

1. Abre `Citas`.
2. Revisa `Mis citas`.
3. Pulsa `Ver detalle` para abrir una cita.
4. Revisa fecha, especialista, centro y duración.
5. Cierra el detalle.

### Cancelar cita propia

1. Abre `Citas`.
2. Localiza una cita `Programada`.
3. Pulsa el botón de cancelar.
4. Confirma con `Cancelar`.

### Consultar solicitudes propias

1. Abre `Citas`.
2. Baja a `Mis solicitudes`.
3. Revisa el estado `Pendiente`, `Aprobada` o `Rechazada`.
4. Revisa la nota de resolución si existe.

### Cancelar solicitud propia

1. Abre `Citas`.
2. Localiza una solicitud `Pendiente`.
3. Pulsa el botón de cancelar.
4. Confirma con `Cancelar`.

### Editar perfil

1. Abre el menú de cuenta.
2. Pulsa `Perfil`.
3. Modifica `Nombre`, `Teléfono` o `Email`.
4. Pulsa `Guardar`.
5. Modifica datos de acceso si procede.
6. Pulsa `Guardar`.
