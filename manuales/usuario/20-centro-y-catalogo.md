## Centro y catálogo

### Seleccionar centro activo

1. Entra como gestor.
2. Abre el selector de centro del menú lateral.
3. Selecciona el centro.
4. Comprueba que aparece como centro activo.

((captura: selector de centro activo))

### Crear centro

1. Abre `Centros`.
2. Pulsa `Centro`.
3. Rellena `Nombre`.
4. Rellena `Ciudad`, si procede.
5. Rellena `Logo`, si procede.
6. Revisa `Horario fijo`.
7. Pulsa `Añadir franja` si necesitas otro hueco.
8. Pulsa `Guardar`.

### Editar centro

1. Abre `Centros`.
2. Pulsa el icono de edición del centro.
3. Edita `Nombre`, `Ciudad` o `Logo`.
4. Ajusta el `Horario fijo`.
5. Pulsa `Guardar`.
6. Pulsa `Volver a centros` para salir.

### Cambiar estado de centro

1. Abre `Centros`.
2. Pulsa la insignia `Activo` o `Inactivo`.
3. Selecciona el nuevo estado.
4. Pulsa `Guardar`.

### Crear servicio

1. Selecciona el centro activo.
2. Abre `Servicios`.
3. Pulsa `Servicio`.
4. Rellena `Nombre`.
5. Rellena `Duración en minutos`.
6. Rellena `Precio`, si procede.
7. Selecciona `Centro`.
8. Selecciona `Especialista`.
9. Rellena `Descripción`, si procede.
10. Pulsa `Guardar`.

### Editar servicio

1. Abre `Servicios`.
2. Pulsa el icono de edición.
3. Modifica los datos.
4. Pulsa `Guardar`.

### Cambiar estado de servicio

1. Abre `Servicios`.
2. Pulsa `Activo` o `Inactivo`.
3. Selecciona el nuevo estado.
4. Pulsa `Guardar`.

### Crear especialista

1. Selecciona el centro activo.
2. Abre `Especialistas`.
3. Pulsa `Especialista`.
4. Rellena `Nombre`.
5. Rellena `Especialidad`, si procede.
6. Selecciona `Estado`.
7. Selecciona `Centro`.
8. Pulsa `Guardar`.

### Editar especialista

1. Abre `Especialistas`.
2. Pulsa el icono de edición.
3. Modifica los datos.
4. Pulsa `Guardar`.

### Registrar ausencia de especialista

1. Abre `Especialistas`.
2. Pulsa el icono `Gestionar ausencias`.
3. Rellena `Inicio`.
4. Rellena `Fin`.
5. Rellena `Motivo`, si procede.
6. Pulsa `Añadir ausencia`.

### Gestionar disponibilidad

1. Edita el centro para cambiar el `Horario fijo`.
2. Abre `Citas`.
3. Cambia a vista `Calendario`.
4. Usa `Día`, `Semana` o `Mes`.
5. Usa una excepción para bloquear una franja o abrir una apertura extra.
