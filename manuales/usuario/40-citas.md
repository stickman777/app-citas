## Citas

### Crear cita desde lista

1. Selecciona el centro activo.
2. Abre `Citas`.
3. Pulsa `Cita`.
4. Selecciona `Cliente`.
5. Selecciona `Servicio`.
6. Selecciona `Especialista`.
7. Selecciona `Fecha y hora`.
8. Si aparecen horas disponibles, pulsa un hueco.
9. Pulsa `Guardar`.

((captura: modal de nueva cita))

### Crear cita desde calendario

1. Abre `Citas`.
2. Pulsa `Calendario`.
3. Selecciona `Día`, `Semana` o `Mes`.
4. Pulsa un hueco disponible.
5. Completa el formulario.
6. Pulsa `Guardar`.
7. Haz doble clic en un hueco no disponible si necesitas una cita fuera de horario.

### Reprogramar cita

1. Abre `Citas`.
2. Pulsa `Lista` o abre la cita desde `Calendario`.
3. Pulsa el icono de edición.
4. Cambia solo `Fecha y hora`.
5. Selecciona un hueco disponible si aparece.
6. Pulsa `Guardar`.

### Editar datos de una cita

1. Abre `Citas`.
2. Pulsa el icono de edición.
3. Cambia `Cliente`, `Servicio` o `Especialista`.
4. No cambies estado en el mismo guardado.
5. Pulsa `Guardar`.

### Cancelar cita

1. Abre `Citas`.
2. Pulsa el icono de cancelar cita.
3. Revisa el cliente afectado.
4. Pulsa `Cancelar cita`.

### Completar cita

1. Abre `Citas`.
2. Pulsa el icono de edición.
3. Cambia `Estado` a `Completada`.
4. Pulsa `Guardar`.
5. No completes una cita que aún no ha comenzado.

### Crear excepción de disponibilidad

1. Abre `Citas`.
2. Pulsa `Calendario`.
3. Abre una excepción existente o crea una desde el calendario.
4. Rellena `Fecha`.
5. Rellena `Hora inicio`.
6. Rellena `Hora fin`.
7. Selecciona `Bloqueado` o `Apertura extra`.
8. Rellena `Etiqueta`, si procede.
9. Pulsa `Guardar`.

### Resolver solicitudes de cita

1. Abre `Solicitudes`.
2. Revisa cliente, servicio, especialista, fecha y motivo.
3. Pulsa `Aprobar` para crear la cita.
4. Añade `Nota para el cliente`, si procede.
5. Pulsa `Aprobar`.
6. Pulsa `Rechazar` si no procede.
7. Añade `Nota para el cliente`, si procede.
8. Pulsa `Rechazar`.
