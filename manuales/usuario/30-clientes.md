## Clientes

### Crear cliente

1. Selecciona el centro activo.
2. Abre `Clientes`.
3. Pulsa `Cliente`.
4. Rellena `Nombre`.
5. Rellena `Teléfono`.
6. Rellena `Email`, si procede.
7. Rellena `Prioridad`, si procede.
8. Selecciona `Centro`.
9. Rellena `Notas`, si procede.
10. Pulsa `Guardar`.

((captura: formulario de cliente))

### Editar cliente

1. Abre `Clientes`.
2. Pulsa el icono de edición.
3. Modifica los datos.
4. Pulsa `Guardar`.

### Activar o desactivar cliente

1. Abre `Clientes`.
2. Pulsa `Activo` o `Inactivo`.
3. Selecciona el nuevo estado.
4. Pulsa `Guardar`.

### Invitar cliente al portal del cliente

1. Abre `Clientes`.
2. Localiza un cliente sin cuenta vinculada.
3. Pulsa `Invitar`.
4. Pulsa `Generar invitación`.
5. Copia el `Enlace de invitación`.
6. Envíalo al cliente por un canal externo.

### Filtrar clientes

1. Usa `Buscar cliente` para texto libre.
2. Abre `Filtro`.
3. Filtra por `Estado`.
4. Filtra por `Cuenta móvil`.
5. Pulsa `Filtrar`.
6. Pulsa `Limpiar todo` para quitar filtros.
