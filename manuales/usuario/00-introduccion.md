# Manual de usuario

## Introducción

### Alcance

- Usa este manual para operar la aplicación web de gestión de citas.
- Usa el área de gestión para centros, servicios, especialistas, clientes y citas.
- Usa el portal del cliente para reservar y consultar citas propias.

### Roles

- Gestor: trabaja en el área de administración.
- Cliente: trabaja en el portal del cliente.
- El backend también define `ADMIN`.
- En este manual, `ADMIN` queda incluido en el uso de gestión cuando accede al área de administración.
