## Backend

### Instalar

1. Entra en la carpeta:

```bash
cd backend
```

2. Instala dependencias:

```bash
npm install
```

### Compilar

```bash
npm run build
```

### Arrancar en desarrollo

```bash
npm run start:dev
```

### Arrancar compilado

```bash
npm run start:prod
```

### Puerto

- Puerto por defecto: `3000`.
- Variable: `PORT`.
- API: `http://localhost:3000`.
- Swagger: `http://localhost:3000/api/docs`.
