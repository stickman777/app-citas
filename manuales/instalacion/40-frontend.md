## Frontend

### Instalar

1. Entra en la carpeta:

```bash
cd frontend
```

2. Instala dependencias:

```bash
npm install
```

### Compilar

```bash
npm run build
```

### Arrancar en desarrollo

```bash
npm run start
```

### Puerto

- URL por defecto: `http://localhost:4200`.
- El backend permite CORS desde `http://localhost:4200` y `http://127.0.0.1:4200`.
- El frontend llama a la API en `http://localhost:3000`.
